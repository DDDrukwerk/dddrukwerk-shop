## 🚀 Quickest Path to Live (30 min)

### 1. Lokaal testen (5 min)
```bash
cd dddrukwerk-shop
npm install
npm run dev
# Open http://localhost:3000
```

### 2. GitHub (5 min)
- Maak account op github.com
- Maak repo `dddrukwerk-shop`
- `git push` je code ernaar

### 3. Vercel (3 min)
- Ga naar vercel.com
- Connect je GitHub repo
- Click "Deploy" → Live!

### 4. Domein (10 min)
- Koop `dddrukwerk.nl` op TransIP
- Connect nameservers naar Vercel
- Wacht 15 min → Done!
